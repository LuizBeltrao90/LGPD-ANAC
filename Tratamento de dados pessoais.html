Privacidade de
Dados Pessoais
📋
Coleta de Dados
Colete apenas informações realmente necessárias para nossas atividades e, sempre que possível, priorize o uso de informações anonimizadas (ex.: CPF XXX.013.123-XX) ou agregadas.
⚖️
Quando Podemos Coletar?
Principalmente para cumprir obrigações legais e regulatórias: fiscalização, certificação, normatização, prestação de serviços, gestão de contratos, relacionamento com a sociedade e gestão de pessoas.
🔄
Tratamento
O tratamento pode envolver diversas etapas. Atenção: a simples visualização já nos torna responsáveis pela proteção dos dados pessoais coletados.
🗄️
Armazenamento
Armazene os dados, preferencialmente, em bancos de dados homologados pela ANAC. Evite guardar dados pessoais em planilhas, e-mails ou meios físicos.
🔴
Dados Sensíveis
Atenção redobrada com dados sensíveis, ou seja, aqueles que possam causar qualquer tipo de discriminação aos seus titulares.
👦
Dados de Crianças e Adolescentes
A coleta de dados de crianças e adolescentes deve ser precedida de consentimento dos pais ou responsáveis e utilizada prioritariamente para fins educacionais, culturais e para a garantia de direitos.
🤝
Compartilhamento
Somente com instituições que ofereçam o mesmo grau de segurança da ANAC e com aprovação formal pela Agência. Compartilhe apenas o estritamente necessário conforme o objeto estabelecido.
🔍
Transparência
O tratamento de dados pessoais deve ser transparente aos titulares em todas as etapas. É um direito assegurado pela LGPD.
🚨
Incidentes
Qualquer incidente deve ser comunicado às unidades competentes na ANAC pelo e-mail: encarregado.lgpd@anac.gov.br
